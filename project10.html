Earthquakes & Fault Lines

Github Repository Project's Website

The purpose of this project is to create a tool that will allow people to visualize the United States Geological Survey's (USGS) earthquake data by using. This app utilizes D3 to import the data from the USGS's website, and Leaflet and Javascript to create maps that display fault lines and information on recent earthquakes, so people can see where earthquakes are most likely to occur.
