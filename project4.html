Stock Market Analysis

Github Repository

Stocks are constantly changing every day, generating an enormous amount of data. The purpose of this analysis is to combine all the data within each stock every year so that users can clearly and easily view the bigger picture of how well a stock did for a particular year. This was completed using Visual Basic for Applications (VBA) to process stock data in an Excel file.
