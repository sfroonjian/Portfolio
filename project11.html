Find Your Destination

Github Repository Project's Website

My group and I (4 people total) created a full-stack travel app containing an interactive map and dashboard for anyone trying to decide which state to travel to. Our webpage provides a choropleth on the most popular states to travel to based on a equation designed by our group. By clicking on each state, users can find information on various attractions, airfare, hotel ratings, and the value of the dollar.

This was created using the following tools:

  • Beautiful Soup and Pandas to scrape the web for data and clean it up
  • An ERD and SQLAlchemy to import the data into PostgreSQL
  • Flask to create dynamic API routes that render Jsonified objects of the data
  • Javascript, D3, Leaflet, and Frappe Charts to display the interactive maps, charts, and lists
  • Heroku to deploy the in the cloud
