World Wide Weather

Github Repository Project's Website

Part I
The purpose of the first part of this project is to determine if there is a correlation between the following:

  1. latitude vs. temperature
  2. latitude vs. cloudiness
  3. latitude vs. humidity
  4. latitude vs wind speeds

This was done by creating code that randomly selects 500-600 cities world-wide, using Open Weather Map API to obtain weather data from each city, and then using Matplotlib to create visualizations of that data.

Part II
The purpose of the second part of this project is use GeoPlaces to create a heat map that displays where the randomly-selected cities are located in the world. The selected cities were narrowed down based on ideal weather conditions determined by the user. All hotels within 5 km of each of the reamining cities were then plotted on a map.
