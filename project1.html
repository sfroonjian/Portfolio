Players in the NBA, NFL, MLB, & NHL

Github Repository

The purpose of this project is to analyze the statistics of players in the NBA, NFL, MLB, and NHL to answer the following questions:

  1. Is player height correlated to ability in the NBA? Is so, what is the optimal height for NBA players?
  2. Is player age correlated to ability in any of the sports? If so, what is the optimal age for each sport?
  3. Are there specific towns/areas that players in each sport are more likely to come from? If so, where?
  4. Are there specific colleges that players in each sport are more likely to attend? Is so, which colleges?

This was done by first importing data from a sports API and an NHL website, cleaning/manipulating the data with Pandas, and extracting only the necessary data. Matplotlib was then used to create ours graphs, and GeoCode was used to find the coordinates of hometowns and colleges. Lastly, the coordinates of hometowns and colleges was used to create a heatmap of each.

  • Python
  • Matplotlib
  • GeoCode
  • APIs
  • Pandas
  • Jupyter Notebook

  • API website used to import data on players of all sports: My Sports Feeds
  • Data on colleges NHL players attended: State of Hockey
