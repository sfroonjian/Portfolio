Predict Your Salary

Github Repository Project's Website

My group and I (5 people total) created a full-stack machine-learning app that predicts the salary of data professionals based on several criteria that the users inputs into a form. In addition to the prediction, we made several charts, graphs, and maps to display various data on the subject including a breakdown of salary by gender, country, and education as well as the most popular technologies used in the industry.

This was created using the following tools:

  • Pandas to extract and transform the data
  • SQLAlchemy to load the data into PostgreSQL
  • SKLearn, specifically Gradient Boosting Regressor, to create and fit the machine learning model
  • SQLAlchemy to query the data stored in PostgreSQL to be used by Flask
  • Flask to render HTML pages, dynamic API routes, and Jsonified objects of the data
  • Jinja to pass data from the Flask app to the HTML pages
  • Natural Language Processing to determine the most commonly used skills in job postings
  • Tableau, Javascript, and D3 to display the interactive charts and maps
  • Heroku to deploy the project

app.py- uses Flask, SQLAlchemy, and Python to connect the PostgreSQL, performs queries of that data, renders API's of JSON's and HTML pages, and inputs user responses into ML model to output a prediction

data_cleaning_sandy.ipynb- uses Python, Pandas, and NumPy to clean and transform the dataset to get it into the desired format to fit the machine learning model

machine_learning_salary_data1.ipynb- uses Pandas and SKLearn to pre-process data, determine feature importances, fit 5 different regression models, and export the best fitting model

model.js- uses JavaScipt and D3 to extract data from API's and create a user-friendly form for users to fill out to see their salary predictions
